오후 12시 즈음 일어났다. 어제 지식 베이스 업데이트 유틸리티가 잘 작동하는 걸 확인했으니 포스트로 작성했다.

지식 베이스 추출 자동화에는 성공했지만 이걸 남들한테 가시적인 성과로 보여주려면 위키 페이지 같은 걸 만들고 싶다.

거기다 지식 베이스를 매번 업데이트할 때 git diff로 비교하는 것도 불편한 일이다.

결국 엑스페리온 본편 세션에 시범삼아 사용하기 전에 지식 베이스 내용을 표시하기 위한 페이지가 필요해졌다.

자동화 도구의 힘을 빌려 이번 패스파인더 50화 업데이트를 빠르게 끝내고 개발에 착수하자.

오후 2시. 지식 베이스 업데이트가 끝났다. 자동화 도구를 사용해서 작업 시간이 1시간 정도 단축된 것 같지만 프롬프트 최적화가 좀 더 필요해보인다.

여전히 사람의 손이 가는 부분이 많다. 기존 내용을 최대한 보존하도록 지시하면 git diff로 처리하기에 조금 더 편해지려나...

개발 중간중간에 있었던 에피소드들을 블로그 포스트로 남겼다. 

인터넷의 개발 블로그 같은 걸 보면 자신이 어떤 프로젝트에 참여했는지 남기는 경우가 종종 있던데, 내 경우는 누군가가 보는 걸 바라는 건 아니고 일기장 겸 나중에 내가 까먹지 않기 위한 용도.

오히려 누군가가 이 블로그를 자세히 들여다보면 그게 더 큰일이다.

이런저런 걸 남기고 나서 오후 5시가 되었다. 생각보다 기록에 시간을 더 써버렸는데, 이것도 내가 살아온 흔적을 일기처럼 남기는 거라고 생각하자.

생각해보니 오늘 아무것도 먹지 않았다. 카페인이 들어가면 배가 덜 고파지는 모양이다.

버거킹을 시키려고 봤는데 언제부터 현장 신용카드 결제가 막혔지? 좋지 않다...

KFC를 시키려고 봤는데 과거에 맛있게 먹었던 블랙 라벨 치킨이 보이지 않고... 내가 1시간쨰 저녁 식사를 고민하고 있는 게 실화인가?

어떻게든 할인 쿠폰을 찾아 KFC의 메뉴를 시키는 걸로 결정. 처음 먹어보는 메뉴인데 괜찮았으면 좋겠다. 업사이드 다운 버거라는 게 과연 어떨까...

술이 좀 땡기는데, 술을 마시기 전 배달 메뉴가 도착하는 시간 전까지 이번 주에 개발할 것을 생각해보자.

구글 AI 스튜디오에서 바이브 코딩을 하는 기능이 있던 것 같은데 Cursor 대신 한 번 사용해볼까?

써보고 나서 영 아니다 싶으면 다시 Cursor로 돌아간다는 생각으로 지식 베이스 위키 페이지 제작을 시켜보기로 했다.

식사가 도착했다. 오늘은 위키 페이지 제작용으로 구글 AI 스튜디오를 가지고 놀다가 술을 마시고 잘 것 같다.

술을 마시다 보니 코어위브의 오퍼링 뉴스가 나왔다. 프리장에 7% 하락까지 본 것 같은데.

아무래도 술을 마실 날을 잘 정한 것 같다.

오늘 술을 마실 생각이 없었더라도 뒤늦게 술을 입에 대고야 말았을 테니까.
