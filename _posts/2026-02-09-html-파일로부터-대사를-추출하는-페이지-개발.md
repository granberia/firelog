TRPG 위키 페이지의 궁극적인 목적은 LLM에 사용하기 위한 프롬프트를 조립해서 완성하는 거다.

각 프로젝트의 log 폴더에 있는 html 파일들을 한 페이지에서 전부 출력시킨 뒤 특정 캐릭터의 이름으로 된 대사를 모두 추출해야 한다.

대사를 추출하는 정규식은 코코포리아 로그 html과 롤20 로그 html을 지원해야 하고, 나중에 TRPG 외의 분야에 쓰려면 사용자가 정규식을 넣을 수도 있게 하자.

이번에도 안티그래비티의 바이브 코딩 기능을 잘 사용해보자.

### 프롬프트

```md
- Sidebar.tsx의 PROMPT와 DOCUMENTS 사이에 WORKSPACE라는 영역을 추가하고 싶어.
- WORKSPACE에는 LOG 라는 이름을 가진 하위 페이지가 존재해야 해.
- 이 페이지는 프로젝트의 log 폴더 내에 존재하는 파일들의 내용을 표시하고, 특정 대사를 추출하기 위한 페이지야.
- 페이지에 진입하면 프로젝트의 log 폴더 내에 존재하는 모든 파일을 접기/펴기가 가능한 리스트로 표시해야 해. 모든 로그 파일은 기본적으로 접힌 상태야.
- 페이지에는 사용자가 선택한 파일들로부터 특정 정규식에 맞는 line들을 추출해서 병합하는 기능이 필요해. 그러므로 정규식을 입력하는 영역과 추출 버튼이 있어야 해.
- 텍스트를 추출할 파일의 구분은 체크박스로 이루어지며, 전부 체크 버튼을 통해 모든 파일을 선택하는 기능도 필요해.
- 정규식을 입력하는 영역에는 기본 값이 들어가있어야 해. campaigns/PER/log/73.html 에서 "라르비온" 이라는 인물이 쌍따옴표를 사용해서 말한 대사가 있는 줄만 추출하도록 기본 값을 설정하자.
- 내용 추출을 마쳤다면 KBPage.tsx 처럼 사용자가 추출된 텍스트들을 모아서 볼 수 있어야 해.
- KBPage.tsx 처럼 Copy to Clipboard 버튼으로 사용자가 추출된 내용을 복사할 수 있어야 해.
```
